## FAQs (5)

1. **ما هي خدمة إدارة برنامج الزي وما يدخل فيها؟** — End-to-end ownership of uniform operations: joiner kit dispatch, leaver-return logistics, scheduled rotation cycles, on-site fittings, inventory dashboard, dedicated programme manager. Handles every uniform touchpoint between procurement signoff and HR-floor execution.
2. **كم تستغرق UNEOM لتسليم زي للموظّف الجديد؟** — 48 hours from HR submitting the new-hire profile via the programme dashboard. Profile includes role, size grade, hijab variant if applicable. Kit dispatches from the city-nearest depot (Riyadh, Jeddah, Dammam Metro) to the new-hire's onboarding location.
3. **هل توفّر UNEOM لوحة لإدارة مخزون الزي؟** — Yes — Pro and Enterprise tiers include the inventory dashboard with: real-time stock levels by role/size, joiner kit dispatch history, leaver-return status, wash-count tracking against rotation schedule, Pantone batch utilisation, and quarterly utilisation analytics.
4. **كيف تُدار عمليات إرجاع زي الموظّفين المغادرين؟** — Exit-kit collection scheduled within 5 working days of HR submitting the leaver record. Each garment is inspected: serviceable units alteration-rebadged for re-issue (cost recovery); end-of-life units sent to the recycling stream (sustainability programme). Reverse-dispatch coordinated through the same depot network.
5. **ما الحدّ الأدنى للموظّفين لخدمة إدارة البرنامج؟** — 200 employees (Standard tier floor). Below 200, Bulk Ordering with quarterly contract typically delivers similar economics without the dashboard layer. Above 2,000, Enterprise tier is recommended for multi-site Pantone coordination and replacement-cycle SLA.
