### 5. The NIJ Body Armor Decision Framework (~500 words across 5 H2 subsections)

- **5.1 NIJ Level IIA** — handgun .9mm + .40 S&W; minimum threshold rarely sufficient for Saudi armed-guarding contexts
- **5.2 NIJ Level II** — handgun .357 Magnum; commonly specified for residential compound + corporate property security
- **5.3 NIJ Level IIIA** — .357 SIG + .44 Magnum FMJ-FN; the Saudi armed-guarding baseline + UNEOM standard carrier rating
- **5.4 NIJ Level III** — rifle 7.62mm; specialised facility security (diplomatic, high-value)
- **5.5 NIJ Level IV** — armor-piercing rifle; specialist tactical only

### 6. The HCIS 2025 Garment Standard Block (~400 words)
- Ripstop construction ≥220 GSM
- Reinforced load-bearing zones (shoulder, elbow, knee)
- Moisture-wicking inner-layer integration
- ANSI/HCIS-aligned hi-vis for outdoor patrol
- Ballistic carrier conformity to NIJ classification matched to threat assessment
- Per-batch documentation requirement

### 7. Comparison Matrix — Tactical Set Tiers (Markdown)

### 8. Decision Framework (~250 words)
"Which NIJ level + HCIS tier?":
- Residential compound + low-risk patrol → Level II + HCIS 2024 baseline
- Corporate complex + diplomatic adjacency → Level IIIA + HCIS 2024 + ANSI 107 Class 2
- Petrochemical / port / high-value facility → Level IIIA + HCIS 2024 + chloride-resistant hardware
- Event ops (Expo 2030, Riyadh Season) → HCIS 2024 baseline + ANSI 107 + breathability emphasis (no carrier unless event security armed-guarding)

### 9. Common Mistakes (~280 words — 4 anti-patterns)

1. **"Ballistic-rated" without NIJ certificate** — unregulated marketing label; failed 2024 audit case (story S-009)
2. **NIJ Level IIA for armed-guarding** — under-spec'd for .357 SIG threat envelope common in Saudi contexts
3. **Single-spec across indoor + outdoor + event** — failed thermal envelope, dust, hi-vis requirements
4. **No annual ballistic carrier re-inspection** — fabric degradation or vest-mount geometry shift over time invalidates initial certification

## Definition Lock-ins (3)

- HCIS 2025 Tactical Garment = ripstop poly-cotton 65/35 ≥220 GSM, reinforced load-bearing zones, moisture-wicking inner-layer, ANSI 107 hi-vis integration, ballistic-carrier-compatible vest-mount geometry
- NIJ Standard-0101.06 = US National Institute of Justice body armor classification, internationally adopted, lab-tested per-Level threat envelope
- "Ballistic-Rated" vs "NIJ-Certified" = the unregulated marketing label vs the lab-attributable classification — only the second is procurement-defensible
