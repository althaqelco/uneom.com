### 5. ISO 14644 Class Decision Block (~500 words across 4 H2 subsections)

- **5.1 Class 5 (ISO Class 5 / FED Class 100)** — sterile pharma manufacturing, sterile-disposable garments mandatory
- **5.2 Class 6–7 (Class 1,000–10,000)** — most pharma + medical device, reusable validated cleanroom suits acceptable
- **5.3 Class 8 (Class 100,000)** — electronics manufacturing, packaging, less stringent garment + reusable standard
- **5.4 Class 9 / non-classified** — controlled environment (lab adjacency), standard institutional uniform with particle awareness

### 6. Gowning Protocol Block (~300 words)
The 8-step gowning sequence for ISO Class 5–7:
1. Pre-gowning anteroom entry + outerwear removal
2. Hand wash + sanitisation
3. Bouffant cap + face mask
4. Cleanroom gown over base layer (validated reusable or sterile single-use)
5. Sterile gloves
6. Cleanroom shoe covers
7. Goggles / face shield where required
8. Final inspection mirror check

### 7. Comparison Matrix — Cleanroom Garment Tiers (Markdown)

| Dimension | Class 5 sterile single-use | Class 5–7 reusable validated | Class 8 standard |
|---|---|---|---|
| Particle shedding | <0.5 µm released particles per cycle | <1.0 µm | <5.0 µm |
| Gamma-sterilisation compatible | Yes | No (autoclave 121°C only) | N/A |
| Cycle count | 1 | 30–50 validated | 100+ standard |
| Cost per use | Highest | Mid | Lowest |
| When mandatory | Sterile pharma fill | Pharma compounding + medical device | Electronics + packaging |

### 8. Decision Framework (~250 words)
"Sterile single-use vs reusable validated vs standard?":
- ISO Class 5 sterile fill → single-use sterile mandatory
- ISO Class 5–7 reusable accepted (with validation pack)
- ISO Class 8 → reusable standard
- Cost per use × annual cycles → ROI break-even between sterile and reusable typically 18–30 months for high-volume contexts

### 9. Common Mistakes (~250 words — 4 anti-patterns)

1. **Standard polyester scrub in Class 7 cleanroom** — particle shedding fails ISO 14644 within 2 wash cycles
2. **No gowning protocol training** — even right garment fails contamination control without operational discipline
3. **Gamma-sterilised garment used past validation cycle count** — fabric integrity degrades; particle shedding rises above threshold
4. **Single-spec Class 5 garment for all manufacturing zones** — over-engineered for Class 8 electronics, wasteful and uncomfortable
